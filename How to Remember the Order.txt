A mnemonic device is a trick that you can use to help you remember something. Use acrostics, or silly sentences, which start with the first initial of each planet name, to help you remember the order, starting with Mercury and ending with Neptune (Pluto has been changed to a 'dwarf planet' so it doesn't count as an actual planet). Some examples are: My Very Easy Method Just Speeds Up Names, or My Very Educated Mother Just Served Us Nachos
